    • Introducción

    es un lenguaje para definir el estilo o la apariencia de las páginas web, escritas con HTML o de los documentos XML. CSS se creó para separar el contenido de la forma, a la vez que permite a los diseñadores mantener un control mucho más preciso sobre la apariencia de las páginas.

    El objetivo inicial de CSS, separar el contenido de la forma, se cumplió ya con las primeras especificaciones del lenguaje. Sin embargo, el objetivo de ofrecer un control total a los diseñadores sobre los elementos de la página ha sido más difícil de cubrir. Las especificaciones anteriores del lenguaje tenían muchas utilidades para aplicar estilos a las webs, pero los desarrolladores aun continúan usando trucos diversos para conseguir efectos tan comunes o tan deseados como los bordes redondeados o el sombreado de elementos en la página.

    Selectores

    Un selector es un conjunto de reglas de estilos aplicadas múltiples elementos de la hoja html. En lugar de aplicar el estilo elemento por elemento, lo aplicamos de manera global.

    Principales selectores

    • Id Se define mediante una # justo antes del nombre de la id. Las reglas de estilo que contiene se aplican al elemento que tenga la id indicada. #id{ /* Estilos */ }
    • Clases Se define mediante un . justo antes del nombre de la clase. Las reglas de estilo que contiene se aplican a cualquier elemento que tenga la clase indicada. Debido a que un elemento puede tener varias clases, también puede recibir estilo de varios selectores de clase. .clase{ /* Estilos */ }
    • Elemento Aplica estilo a todos los elementos que tengan la etiqueta html indicada. Por ejemplo, un selector del elemento div aplicaría estilos a todos los div. div{ /* Estilos */ }
    Variables

    Declaración y asignación: --nombrevariable: value;

    Utilización: propiedad: var(--nombrevariable, fallback);

  • El fallback no es necesario. Se utiliza para que el navegador tenga un respaldo en caso de que la variable sea inválida. Es útil para debuggear.
  • Otra manera de hacer respaldos es poner la misma propiedad justo antes con un valor con más soporte (aprovechando que, en caso de igual importancia, la prioridad va de abajo hacia arriba).
    •

    Las variables en CSS también tienen efecto cascada. Por tanto, son utilizables dentro del elemento en el que han sido creadas y de cualquiera de los elementos que estén anidados dentro. Por esto, las variables CSS suelen declararse en la pseudoclase :root (contenedor de todo el documento .html).

    Accesibilidad

    Elementos visibles sólo para lectores de pantalla Esto es útil para mantener elementos que son útiles para el lector de pantalla pero que no queremos que sean visibles para la mayoría de usuarios, como por ejemplo el h1.

    .sr-only { position: absolute; left: -10000px; width: 1px; height: 1px; top: auto; overflow: hidden; }

    Ratio de contraste y daltonismo Las WCAG recomiendan que el ratio de contraste de texto respecto a fondo sea de por lo menos 4,5:1. Se debe evitar destacar texto exclusivamente mediante tono mientras se mantienen saturación o luminosidad similares a las del contenido que lo rodea, o diferenciar fondo de primer plano sólo por luminosidad o saturación (sin cambiar o cambiando poco el tono).

    Media queries

    Permiten cambiar los valores de las reglas de un selector en función del tamaño del viewport. Se utilizan para hacer contenido adaptativo. También pueden alterar el valor de las variables.

    @media (max-width: 350px) { :root { --penguin-size: 200px; } p { color: red; } }

    Esta media querie se activaría si el ancho del viewport es menor o igual a 350 píxeles.
